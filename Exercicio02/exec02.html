Exercicio sobre TAGs - O que cada TAGs que eu aprendi faz:

  • DOCTYPE

    • A TAG !DOCTYPE mostra qual formato de arquivo será usado

  • html

    • A Tag html serve para colocar os códigos que aparecerão dentro da página

  • head

    • A TAG head serve para colocar-mos os códigos que aparecerão na "cabeça" da nossa página

  • title

    • Essa TAG serve para colocar um título para nossa página

  • style

    • Esta TAG serve para modificar o estilo da página e dos itens contidos nela

  • body

    • TAG usada para colocar-mos o que vai aparecer no "corpo" da nossa página

  • h1-h6

    • TAG usada para definir o tamanho de um título sendo h1 maior e h6 menor

  • ul

    • TAG usada para definir um espaço do parágrafo

  • li

    • TAG para criar bolinha no começo de um paragrafo
